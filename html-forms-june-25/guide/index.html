üß† Understanding Regex Patterns

üîç Regex Cheat Sheet:

  • [0-9] ‚Üí Any digit
  • [a-z] ‚Üí Lowercase letter
  • [A-Z] ‚Üí Uppercase letter
  • {3} ‚Üí Exactly 3 times
  • {3,} ‚Üí 3 or more times
  • {3,10} ‚Üí 3 to 10 times
  • ? ‚Üí Optional (0 or 1)
  • + ‚Üí One or more
  • * ‚Üí Zero or more
  • \+ ‚Üí Literal + symbol
  • (?=.*[A-Z]) ‚Üí Must contain uppercase
  • ^ ‚Üí Start of string
  • $ ‚Üí End of string
