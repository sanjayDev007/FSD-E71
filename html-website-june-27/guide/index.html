üéõÔ∏è Powerful Table Attributes for Layouts

colspan

Makes a cell span multiple columns - perfect for headers and footers!

<td colspan="3" bgcolor="#1565c0">Header</td>

rowspan

Makes a cell span multiple rows - great for sidebars!

<td rowspan="2" width="200">Sidebar</td>

width

Controls column width for perfect layout control

<td width="30%">30% width</td>

bgcolor

Background color for visual sections

<td bgcolor="#e3f2fd">Light blue background</td>

üí° Essential Table Attributes

  • cellpadding - Space inside cells (like padding)
  • cellspacing - Space between cells
  • border - Table border thickness
  • width - Table or cell width (pixels or %)
  • align - Horizontal alignment (left, center, right)
  • valign - Vertical alignment (top, middle, bottom)
  • bgcolor - Background color

‚ùå Common Table Layout Mistakes

  • Not setting table width properly
  • Forgetting cellpadding and cellspacing
  • Missing valign for proper alignment
  • Not using colspan/rowspan effectively
  • Inconsistent bgcolor usage

‚úÖ Pure HTML Table Best Practices

  • Use width="100%" for full layouts
  • Set cellpadding="10" for spacing
  • Use valign="top" for content alignment
  • Apply bgcolor for visual separation
  • Use border="0" for clean layouts
